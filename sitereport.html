Site Development Report

Development Experience

As a second-semester BSc Web Development and Cyber Security student, this portfolio project represents my growing understanding of web development principles. I'm focusing on creating secure and efficient web solutions while learning the fundamentals of HTML, CSS, and responsive design.

My academic background in cyber security has made me particularly mindful of implementing secure practices in web development. This portfolio showcases both my design capabilities and my commitment to security best practices.

Design Decisions

The design approach focused on minimalism and user experience. I chose a color scheme based on blues and neutral tones to convey professionalism while maintaining visual interest. The primary colors used are:

  • Primary Blue (#3498db) - For highlighting and calls-to-action
  • Dark Blue (#2c3e50) - For headers and navigation
  • Light Gray (#f5f7fa) - For backgrounds

The font selection prioritized readability, using Arial as the primary font with appropriate fallbacks. The spacing and typography were influenced by modern web design trends seen on platforms like Dribbble and Behance.

The gradient background adds subtle depth while maintaining content readability. Content cards feature slight shadows and rounded corners for a modern, clean aesthetic.
