Site Development Report

Development Experience

As you know that I'm currently in my second semester of BSc Web Development and Cyber Security, I don't know the advanced principles of Web Development and Cyber Security. Thus, this portfolio project basically represents my journey through this degree; about how I'm grasping the fundamentals of Web Development and how I'm improving so far. I'm currently focusing on the basic principles of HTML and CSS through which I've created this portfolio website.

Since we had to create and also design the website on our own, I had to first think of what the design should be like. The font, the design, the colours and so on. I decided to be minimalistic and follow the current website design trends. This portfolio basically shows both my design and my implementation capabilities.

Design Decisions

Mainly, the design is based on simplicity and to enhance user experience. The design follows a basic principle and that is; a navigation bar on the top, a context box where the content is written, and everything is circular or plain abstract. The colours that I chose are based on blues and neutral tones as they portray simplicity and professionalism. The primary colors used are:

  • Primary Blue (#3498db) - For highlighting and calls-to-action
  • Dark Blue (#2c3e50) - For headers and navigation
  • Light Gray (#f5f7fa) - For backgrounds

I used Arial as my primary font with similar fallbacks. Why? because this font as easy to read and is extremely user friendly and modern. The spacing and the website text layout was mainly influenced by modern design patterns that are used in platforms like Dribbble, Canva, etc.

The entire website has a gradient background with a horizontal blueish color sliding past the page. This adds a sleek touch to the website. The content boxes also have a slight shadow behind them which adds to the depth of the website and they have rounded corners as well to portray modern aesthetics.
